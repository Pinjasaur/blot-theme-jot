# [Blot][blot] Theme: Jot

This is my [personal theme][blog] for [Blot][blot]. It's a fork of the Index theme.

## Features

- Light and dark theme(!!)
- Hash `id`s for each header
- Normalized styling across browsers (removes horizontal scrollbar)
